Web Development

Project Overview
Build a white-labeled platform to allow organizations to collect donations from users using the Stripe API.

Challenges
The main challenge of this project was that the client wanted to white label the application to many different organizations and so the code had to work across multiple URLs. The solution was to host the front-end code (React) in separate S3 buckets and re-direct users using CloudFront. The same back-end (Django) was used for each website and we used EC2 to host it.

Technologies
Django, React, Stripe, Material UI, AWS (S3, CloudFront, EC2)
